#todo.py

a To Do CLI application with SQLite backend

###INSTALLATION

```sh
python setup.py install
```


###USAGE

####New todo

```sh
todocli -n "Learn Colmak"
```

####New todo with priority
```sh
todocli -n "Learn Colmak" -p 10
```

Add new todo item without deadline. You can use --new also.
